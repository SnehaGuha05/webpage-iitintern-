Evaluating Large Language Models as Virtual Annotators for Physical Sensing Data

Virtual Annotators for Physical Sensing Data

This project explores the use of Large Language Models (LLMs) as virtual annotators for physical sensing data. By combining self-supervised sensor representations with LLM reasoning, we aim to replace traditional annotation pipelines that rely on costly human effort and auxiliary modalities such as video or audio. Evaluations on benchmark human activity recognition datasets show that LLMs can generate accurate labels from encoded sensor data, offering a scalable, privacy-preserving, and efficient alternative to conventional annotation methods.

Key Contributions

This work investigates the feasibility of using Large Language Models (LLMs) as virtual annotators for time-series physical sensing data and proposes a novel framework for improving annotation quality through representation learning. The key contributions of this work are summarized as follows:

  1. Exploring LLM’s Capability as Virtual Annotators of Physical Sensing Data. We perform a thorough analysis to investigate whether state-of-the-art LLMs such as GPT-4 can function as virtual annotators for physical sensing data, using locomotive data as a representative use case for human mobility annotation. Our study demonstrates that while LLMs can comprehend locomotive sensor signatures such as triaxial accelerometer data, they struggle to accurately annotate raw sensor streams with the correct mobility classes, even when contextual information and bootstrap examples are provided in the prompt.
  2. Extending LLM’s Capability as Virtual Annotators of Physical Sensing Data. Based on the findings from the pilot study, we develop an approach that encodes locomotive sensor data into an embedding space capable of capturing class-specific signatures. These embeddings provide rich contextual information to LLMs and enable more effective annotation of accelerometer signatures. We explore self-supervised contrastive learning using two configurations: (a) time-domain signatures only and (b) combined time- and frequency-domain signatures. This work introduces a novel approach for enhancing the annotation capability of pre-trained LLMs without requiring computationally expensive fine-tuning or sophisticated prompt engineering.
  3. Thorough Evaluation, Analysis, and Discussion. We conduct extensive experiments on four benchmark Human Activity Recognition (HAR) datasets to analyze how encoded accelerometer representations improve LLM reasoning and annotation accuracy under varying numbers of examples and activity classes. Beyond accuracy, we also evaluate practical considerations such as token consumption, API rate limits, processing time, and monetary cost. The study summarizes the trade-offs involved, highlights the benefits of using LLMs as virtual annotators, and discusses the limitations of relying solely on unimodal triaxial accelerometer data, particularly in scenarios involving multiple activity classes.
